حماية مفتاح الـ API | API Key Security

التعليمات Instructions

١. لا تشارك المفتاح — Don't share the key

المفتاح شخصي جدًا، إذا شاركته ممكن يتم استغلاله.
The API key is private. Sharing it lets others use your account.

٢. لا تخزنه في localStorage — Don't store in localStorage

localStorage غير آمن ويمكن سرقته عبر JavaScript خبيث.
localStorage is unsafe and can be stolen by malicious scripts.

٣. استخدم مدير كلمات مرور — Use a Password Manager

مدير كلمات المرور يحفظ مفاتيحك بشكل مشفر.
A password manager keeps your keys encrypted and safe.

٤. غيّر المفتاح بانتظام — Rotate your key regularly

تغيير المفتاح يقلل خطر الاستغلال.
Changing the key reduces the risk of misuse.

٥. احذر من مواقع مشبوهة — Avoid suspicious sites

لا تضع المفتاح في مواقع مجهولة أو غير رسمية.
Never enter your key on unknown or unofficial sites.
